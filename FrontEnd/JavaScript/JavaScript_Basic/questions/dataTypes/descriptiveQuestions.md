### In how many ways we create string primitive data type in JavaScript?

1. Using string literals

```javascript
    let string1="hello world";
```

2. Using the `String`  constructor for primitive values

```javascript
    let str1 = String("hello world");
```

3. Using template literals

```javascript
    let name = "mohan";
    let str1 = `Hello ${name}`;
```

4. Using the `String()` function

```javascript
    let str1 = String(12345);
```

### What is the difference between string literal and string primitive data type in JavaScript?

String literals and string primitive data types are the same in JavaScript. Both are used to represent a sequence of characters.

### How to check if a value is string primitive data type in JavaScript?

You can use the `typeof` operator to check if a value is a string primitive data type.

```javascript
    let str1 = "hello world";
    console.log(typeof str1);  // Output: string
```

### What is the use of length property in string primitive data type in JavaScript?

The `length` property returns the number of characters in a string.

```javascript
    let str1 = "hello world";
    console.log(str1.length);  // Output: 11
```

### How to access characters in a string primitive data type in JavaScript?

You can access characters in a string using the `charAt()` method or by using square brackets `[]`.

```javascript
    let str1 = "hello world";
    console.log(str1.charAt(0));  // Output: h
    console.log(str1[0]);  // Output: h
```
